## Description

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```

## Seed

```bash
# drop db
$ pnpm db:drop

# migrate db
$ pnpm db:migrate

# seed data
$ pnpm db:seed
```
